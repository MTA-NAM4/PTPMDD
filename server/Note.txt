Cài đặt JSON server
1. 
Download node js:  http://nodejs.org/en/
Chọn phiên bản Long Term Support (LTS)
Chạy file đã download để cài đặt
2. 
Chạy Command Prompt
Gõ node -v
Gõ npm -v
Gõ npm install -g json-server
//=======================================================
1. Start Server: 
json-server --watch db.json

2. truy cập: 
http://10.0.2.2:3000/people


btvn: từ client thêm 1 item vào file trên server db.json
